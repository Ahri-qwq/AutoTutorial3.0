# 参考文章说明

## 目录结构

```
data/reference_materials/
└── style_references/          # 风格参考文章
    ├── 文章1.md
    ├── 文章2.md
    └── ...
```

## 用途

这个文件夹用于存放优秀的教程文章，作为写作风格参考。

Claude Code在执行Step 1（知识调研）时会：
1. 读取这个文件夹中的3-5篇参考文章
2. 提取风格特征：
   - 开头方式（如何引入主题）
   - 结构偏好（章节组织方式）
   - 语言特征（专业但易懂的表达）
   - 代码呈现（代码块格式、参数说明方式）
3. 在后续撰写和风格审查时参考这些特征

## 文章要求

### 数量
- 建议：3-5篇
- 最少：2篇
- 最多：不超过10篇（太多会增加处理时间）

### 格式
- 推荐：Markdown格式（.md）
- 也支持：纯文本（.txt）
- 不建议：docx、pdf（需要额外处理）

### 内容
- 必须是ABACUS或第一性原理计算相关的教程文章
- 内容质量高、结构清晰
- 语言风格是你希望模仿的
- 包含代码示例和参数说明

### 示例
好的参考文章应该包含：
- 清晰的章节结构
- 物理背景介绍
- 参数详细说明
- 完整的代码示例
- 实际案例演示

## 如何添加文章

### 方法1：直接复制
将文章文件复制到这个文件夹：
```bash
cp your_article.md data/reference_materials/style_references/
```

### 方法2：从网页保存
如果文章在网页上：
1. 复制文章内容
2. 保存为.md或.txt文件
3. 放到这个文件夹

## 注意事项

1. **文件命名**
   - 使用有意义的文件名
   - 例如：`ABACUS弹性常数计算教程.md`
   - 避免：`文章1.md`、`temp.md`

2. **文件编码**
   - 使用UTF-8编码
   - 避免使用GBK等其他编码

3. **文件大小**
   - 单个文件不要太大（建议<100KB）
   - 如果文章很长，可以只保留核心部分
