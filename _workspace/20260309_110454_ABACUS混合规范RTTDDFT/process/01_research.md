# 知识调研报告

## 1. RAG 检索结果

### 关键参数（已确认）
- `td_stype = 2`：混合规范（Hybrid gauge），来源：ABACUS input_main 官方文档
  - `0` = 长度规范（非周期体系）
  - `1` = 速度规范（周期体系，但 NAO 下有系统性误差）
  - `2` = 混合规范（周期与非周期均适用，NAO 误差最小）
- 参考文献：*J. Chem. Theory Comput.* 2025, 21, 3335−3341（赵昊天，何力新）

### TDDFT 完整 INPUT 参数
必须设置：
- `calculation md`：必须为 md
- `esolver_type tddft`：TDDFT 求解器
- `basis_type lcao`：必须为 LCAO（NAO 基组）
- `gamma_only 0`：rt-TDDFT 需要 complex 类型，不兼容 gamma_only=1

时间步设置：
- `md_nstep`：总时间步数（建议 >400步 以保证 >20 fs 演化，md_dt=0.05时）
- `md_dt`：时间步长（单位 fs，通常取 0.02–0.05）
- `md_tfirst 0`：离子初始温度为 0（固定离子）

电场设置：
- `td_vext 1`：开启外加电场
- `td_stype 2`：混合规范（本教程重点）
- `td_tstart 1`：第 1 步开始加电场
- `td_tend <N>`：最后加电场的步数
- `td_vext_dire <方向>`：1=x, 2=y, 3=z
- `td_ttype 0`：高斯型脉冲（E(t) = A·cos[2πf(t-t0)+φ]·exp[-(t-t0)²/2σ²]）

高斯型脉冲参数（td_ttype=0 时）：
- `td_gauss_freq`：频率（单位 eV/2π?，实际取决于 ABACUS 内部单位）
- `td_gauss_phase`：相位
- `td_gauss_sigma`：高斯宽度σ（单位 fs）
- `td_gauss_t0`：脉冲中心时刻（步数）
- `td_gauss_amp`：振幅（电场强度，单位 V/Å）

输出开关：
- `out_dipole 1`：输出电偶极矩（非周期体系/长度规范用）
- `out_current 1`：输出电流（周期体系/速度规范/混合规范）
- `out_efield 1`：输出外加电场时程

### 后处理
- 吸收谱：α(ω) = FT[D(t)] / FT[E(t)]（电偶极矩方法）
- 介电函数：电流 → 极化 → 介电函数（周期体系）
- 高次谐波谱（HHG）：对强场激发下的电流信号做傅里叶变换
- 工具：`tools/plot-tools` 目录下有后处理脚本

## 2. 案例解析

### 案例体系
**体系1：HCO 分子**（非周期体系，用于方法验证）
- 外场：高斯脉冲
- 输出：响应电流 + 吸收谱
- 结论：长度规范 ≈ 混合规范 >> 速度规范（低频发散）

**体系2：Si 单胞**（周期体系，核心应用）
- 弱场（δ脉冲激发）：响应电流 → 介电函数
  - 结论：混合规范 > 速度规范（有系统误差）
- 强场（高斯脉冲，场强从 0.001 到 0.01 V/Å 递增）：高次谐波谱（HHG）
  - 谐波阶次：1、3、5、7次谐波依次出现

### 案例特点
- 无直接 INPUT/STRU 文件（公众号文章），需根据 TDDFT 文档构造示例
- 核心参数：`td_stype = 2` 是混合规范的开关
- 对比实验：三种规范的计算结果对比是案例的核心内容
