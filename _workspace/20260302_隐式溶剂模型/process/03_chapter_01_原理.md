# 第一章：隐式溶剂模型原理

## 1.1 为什么需要溶剂模型

第一性原理计算默认在真空中进行。真实体系往往不是如此——电催化反应、固液界面、溶液相分子都处于液体环境中。溶剂分子会与溶质发生相互作用，改变体系的电荷分布和总能量，这种效应称为**溶剂化效应**。

忽略溶剂化效应时，计算得到的吸附能、反应自由能等物理量往往与实验偏差较大。

## 1.2 显式溶剂 vs 隐式溶剂

处理溶剂化效应有两类方法：

**显式溶剂（Explicit Solvation）**：在超胞中逐个放置溶剂分子（如水分子），直接参与 DFT 计算。优点是精度高，缺点是计算量随溶剂分子数急剧增加，且需要对溶剂构型进行充分采样。

**隐式溶剂（Implicit Solvation）**：将溶剂视为均匀的连续介质，用介电常数描述其极化响应，而不显式放置溶剂分子。溶剂自由度被"平均化"到介质的宏观性质里。计算量与真空计算相当，适合大型体系和需要扫描多个构型的研究。

两者不互斥——隐式溶剂给出定性正确的结果，计算成本低；显式溶剂更精确但更昂贵。实际研究中常以隐式溶剂快速筛选，再用显式溶剂精算关键构型。

## 1.3 ABACUS 中的实现：Mathew 2014 方法

ABACUS 实现的隐式溶剂模型采用了 Mathew、Sundararaman、Letchworth-Weaver、Arias 和 Hennig 在 2014 年提出的方法 [1]。其核心思想是：

**介电腔**：根据溶质的电子密度分布，自动划定溶质-溶剂的界面。电子密度高于阈值 `nc_k` 的区域为溶质（真空，ε = 1），低于该阈值的区域为溶剂（介电常数 ε = `eb_k`）。界面区域的过渡由扩散宽度参数 `sigma_k` 控制，使腔壁平滑而非突变。

**能量修正的两个分量**：在溶剂化计算中，总能量在真空 SCF 的基础上增加两项修正：

- **`E_sol_el`（静电项）**：溶质的电荷在连续介质中诱导极化，极化场反过来作用于溶质，产生稳定化效应。对于带电荷分布的溶质，该项通常为**负值**（降低体系能量）。

- **`E_sol_cav`（空腔项）**：溶质在溶剂中"挤出"一个空腔，需要克服溶剂分子间的范德华力和表面张力，这一过程耗能。该项通常为**正值**（升高体系能量）。空腔项由参数 `tau`（有效表面张力）控制。

溶剂化能量修正 = `E_sol_el` + `E_sol_cav`。在水溶液中，极性溶质的静电稳定化通常大于空腔能量代价，净效果是溶剂化降低体系总能量。

以本教程的 H₂ 分子案例为例，H₂ 极性弱，溶剂化效应较小，但两个分量的符号和量级体现了典型的溶剂化行为，是理解模型的好入口。
