# 前言

本教程介绍如何在 ABACUS 中使用**隐式溶剂模型（Implicit Solvation Model）**，对处于液体环境中的体系进行第一性原理计算。

## 教程目标

学完本教程，你将能够：
- 理解隐式溶剂化模型的基本物理思想
- 在任意 ABACUS 计算中开启溶剂化效应
- 读懂输出中的 `E_sol_el` 和 `E_sol_cav`，并计算体系的溶剂化能

## 适用读者

- 有 ABACUS 基础（能独立运行 SCF 计算）
- 需要模拟电催化、固液界面或溶液相反应的用户

# 第一章：隐式溶剂模型原理

## 1.1 为什么需要溶剂模型

第一性原理计算默认在真空中进行。真实体系往往不是如此——电催化反应、固液界面、溶液相分子都处于液体环境中。溶剂分子会与溶质发生相互作用，改变体系的电荷分布和总能量，这种效应称为**溶剂化效应**。

忽略溶剂化效应时，计算得到的吸附能、反应自由能等物理量往往与实验偏差较大。

## 1.2 显式溶剂 vs 隐式溶剂

处理溶剂化效应有两类方法：

**显式溶剂（Explicit Solvation）**：在超胞中逐个放置溶剂分子（如水分子），直接参与 DFT 计算。优点是精度高，缺点是计算量随溶剂分子数急剧增加，且需要对溶剂构型进行充分采样。

**隐式溶剂（Implicit Solvation）**：将溶剂视为均匀的连续介质，用介电常数描述其极化响应，而不显式放置溶剂分子。溶剂自由度被"平均化"到介质的宏观性质里。计算量与真空计算相当，适合大型体系和需要扫描多个构型的研究。

两者不互斥——隐式溶剂给出定性正确的结果，计算成本低；显式溶剂更精确但更昂贵。实际研究中常以隐式溶剂快速筛选，再用显式溶剂精算关键构型。

## 1.3 ABACUS 中的实现：Mathew 2014 方法

ABACUS 实现的隐式溶剂模型采用了 Mathew、Sundararaman、Letchworth-Weaver、Arias 和 Hennig 在 2014 年提出的方法 [1]。其核心思想是：

**介电腔**：根据溶质的电子密度分布，自动划定溶质-溶剂的界面。电子密度高于阈值 `nc_k` 的区域为溶质（真空，ε = 1），低于该阈值的区域为溶剂（介电常数 ε = `eb_k`）。界面区域的过渡由扩散宽度参数 `sigma_k` 控制，使腔壁平滑而非突变。

**能量修正的两个分量**：在溶剂化计算中，总能量在真空 SCF 的基础上增加两项修正：

- **`E_sol_el`（静电项）**：溶质的电荷在连续介质中诱导极化，极化场反过来作用于溶质，产生稳定化效应。对于带电荷分布的溶质，该项通常为**负值**（降低体系能量）。

- **`E_sol_cav`（空腔项）**：溶质在溶剂中"挤出"一个空腔，需要克服溶剂分子间的范德华力和表面张力，这一过程耗能。该项通常为**正值**（升高体系能量）。空腔项由参数 `tau`（有效表面张力）控制。

溶剂化能量修正 = `E_sol_el` + `E_sol_cav`。在水溶液中，极性溶质的静电稳定化通常大于空腔能量代价，净效果是溶剂化降低体系总能量。

以本教程的 H₂ 分子案例为例，H₂ 极性弱，溶剂化效应较小，但两个分量的符号和量级体现了典型的溶剂化行为，是理解模型的好入口。
# 第二章：算例获取与文件结构

## 2.1 算例下载

ABACUS 官方提供了隐式溶剂模型的完整算例（以 Pt 表面板层为例），可从以下地址获取：

下载完成后进入：
```

```

**国际（GitHub）：**
```
https://github.com/deepmodeling/abacus-develop/tree/develop/examples/implicit_solvation_model/Pt-slab
```

本教程的理论演示案例使用的是更简单的 **H₂ 分子体系**（`suffix = H2`），该体系将一个氢分子置于超胞中，模拟其处于水溶液中的状态。这一体系计算快、结构简单，便于聚焦在溶剂化参数的理解上。

## 2.2 PW 基组计算的文件结构

使用平面波（PW）基组进行隐式溶剂 SCF 计算，需要以下文件：

```
计算目录/
├── INPUT          # 计算参数（本教程重点）
├── STRU           # 晶胞和原子坐标
├── KPT            # k 点设置
└── H_ONCV_PBE-1.0.upf   # H 原子赝势文件
```

> 注意：PW 基组无需轨道文件（`.orb`），这是与 LCAO 基组的重要区别。

## 2.3 体系说明：H₂ 分子在超胞中

本案例的物理图像：将一个孤立的 H₂ 分子放在一个足够大的超胞中，四周被隐式溶剂（水）包围。超胞需要足够大，以避免周期性镜像之间的相互作用。

INPUT 文件中的几个参数直接对应这一体系：

- `ntype = 1`：体系只有一种元素（H）
- `nbands = 2`：H₂ 有 2 个价电子，对应 2 条能带
- `symmetry = 0`：关闭晶体对称性（分子体系一般不用晶体对称性）
- `pseudo_dir = ./`：赝势文件放在当前目录下

STRU 文件需要定义超胞大小和 H₂ 的原子坐标。对于孤立分子，超胞通常取立方盒子，边长约 10-15 Å。KPT 文件只需取 Γ 点（`1 1 1 0 0 0`），分子体系不需要 k 点采样。具体文件格式可参考官方 Pt-slab 算例，按分子体系调整。
# 第三章：INPUT 参数解析

## 3.1 完整 INPUT 文件

以下是本教程案例的完整 `INPUT` 文件：

```
INPUT_PARAMETERS
#Parameters (1.General)
suffix              H2
calculation         scf
ntype               1
nbands              2
symmetry            0
pseudo_dir          ./

#Parameters (2.Iteration)
ecutwfc             60
scf_thr             1e-6
scf_nmax            100

#Parameters (3.Basis)
basis_type          pw

#Parameters (Solvation Model)
imp_sol             1
eb_k                80
tau                 0.000010798
sigma_k             0.6
nc_k                0.00037
```

使用隐式溶剂模型，只需在原有 INPUT 的基础上添加 `#Parameters (Solvation Model)` 部分即可，其余参数与常规 SCF 计算相同。

## 3.2 通用参数说明

| 参数 | 案例值 | 说明 |
|------|--------|------|
| `suffix` | `H2` | 输出目录前缀（`OUT.H2/`） |
| `calculation` | `scf` | 自洽场计算 |
| `ntype` | `1` | 元素种类数（本例只有 H）|
| `nbands` | `2` | 计算能带数（H₂ 有 2 个价电子）|
| `symmetry` | `0` | 关闭晶体对称性 |
| `pseudo_dir` | `./` | 赝势文件在当前目录 |
| `ecutwfc` | `60` | 平面波截断能，单位 Ry |
| `scf_thr` | `1e-6` | 电荷密度自洽收敛阈值，单位 Ry |
| `scf_nmax` | `100` | SCF 最大迭代步数 |
| `basis_type` | `pw` | 使用平面波基组 |

> `ecutwfc = 60 Ry` 对氢元素是较充分的截断能。实际计算时建议先做截断能收敛测试。

## 3.3 溶剂化参数详解

### `imp_sol`：溶剂模型开关

```
imp_sol             1
```

- **类型：** Bool
- **取值：** `1`（开）或 `0`（关）
- **默认值：** `0`
- **说明：** 设为 `1` 时启用隐式溶剂模型，在 SCF 迭代中加入溶剂化能量修正。

---

### `eb_k`：溶剂相对介电常数

```
eb_k                80
```

- **类型：** Real
- **默认值：** `80`（水的相对介电常数）
- **说明：** 描述溶剂对电场的屏蔽能力。介电常数越大，溶剂对溶质电荷的极化响应越强，静电稳定化效果越显著。

常见溶剂的参考介电常数（近似值，实际以实验数据为准）：

| 溶剂 | eb_k（近似值） |
|------|---------------|
| 水（Water） | 80 |
| 乙醇（Ethanol） | 24.5 |
| 乙腈（Acetonitrile） | 37.5 |
| 二甲亚砜（DMSO） | 46.7 |

> **注意：** 只需更改 `eb_k` 并不足以完整模拟非水溶剂。`tau`、`sigma_k`、`nc_k` 三个参数的默认值是针对水拟合的，用于非水溶剂时需重新拟合。

---

### `tau`：有效表面张力参数

```
tau                 0.000010798
```

- **类型：** Real
- **单位：** Ry/Bohr²
- **默认值：** `1.0798e-05`（即 `0.000010798`，与案例值一致）
- **说明：** 控制空腔能量项 `E_sol_cav` 的大小。物理上对应溶质在溶剂中形成空腔所需克服的有效表面张力，捕获了溶质-溶剂之间的空化、色散和排斥相互作用。

案例使用默认值，适用于水溶剂。

---

### `sigma_k`：扩散腔宽度

```
sigma_k             0.6
```

- **类型：** Real
- **单位：** 无量纲
- **默认值：** `0.6`
- **说明：** 控制溶质-溶剂界面的过渡宽度。`sigma_k` 越大，腔壁越"软"（过渡越宽）；越小，界面越陡。该参数与 `nc_k` 联动，共同决定介电腔的形状。

---

### `nc_k`：介电腔形成的电子密度阈值

```
nc_k                0.00037
```

- **类型：** Real
- **单位：** Bohr⁻³
- **默认值：** `0.00037`
- **说明：** 电子密度高于此值的区域被认为是"溶质内部"（ε = 1），低于此值的区域被认为是"溶剂"（ε = `eb_k`）。该阈值决定了介电腔的大小：阈值越小，腔越大。

## 3.4 溶剂化参数速查表

| 参数 | 类型 | 案例值 | 默认值 | 单位 | 是否需要调整（非水溶剂）|
|------|------|--------|--------|------|------------------------|
| `imp_sol` | Bool | 1 | 0 | — | 否（开关） |
| `eb_k` | Real | 80 | 80 | — | **是**（必须改） |
| `tau` | Real | 0.000010798 | 1.0798e-05 | Ry/Bohr² | 建议重新拟合 |
| `sigma_k` | Real | 0.6 | 0.6 | 无量纲 | 建议重新拟合 |
| `nc_k` | Real | 0.00037 | 0.00037 | Bohr⁻³ | 建议重新拟合 |

**对于水溶剂：** 案例中所有 5 个参数均使用（或等同于）默认值，直接复制即可。

**对于非水溶剂：** 至少修改 `eb_k`；若需要定量精度，`tau`/`sigma_k`/`nc_k` 需针对目标溶剂与实验数据拟合。
# 第四章：运行与结果分析

## 4.1 运行计算

准备好所有输入文件（INPUT、STRU、KPT 及赝势文件）后，在计算目录下运行：

```bash
# 串行运行
abacus

# 并行运行（以 4 核为例）
OMP_NUM_THREADS=1 mpirun -np 4 abacus
```

计算完成后，输出文件位于 `OUT.H2/` 目录（对应 `suffix = H2`）。

## 4.2 在输出文件中找到溶剂化能量修正

打开 `OUT.H2/running_scf.log`，搜索 `E_sol_el` 和 `E_sol_cav`：

```bash
grep "E_sol" OUT.H2/running_scf.log
```

在每次 SCF 迭代中都会输出当前的溶剂化能量修正，最终收敛值出现在 SCF 完成后的汇总区域。

两个量的含义：

| 输出量 | 物理含义 | 通常符号 |
|--------|----------|----------|
| `E_sol_el` | 溶质-溶剂静电感应能 | 负值（稳定化） |
| `E_sol_cav` | 溶质在溶剂中形成空腔的能量代价 | 正值（不稳定化） |

对于 H₂ 分子这样的弱极性体系：
- `E_sol_el` 较小（H₂ 本身极性弱，溶剂的静电稳定化有限）
- `E_sol_cav` 同样较小（H₂ 体积小，占据的空腔小）

## 4.3 总能量与溶剂化能计算

SCF 完成后，`running_scf.log` 中同样输出总能量：

```bash
grep "FINAL_ETOT_IS" OUT.H2/running_scf.log
```

**溶剂化能的计算方法**：

溶剂化能 = E(imp_sol=1) − E(imp_sol=0)

具体操作：
1. 运行一次 `imp_sol = 1` 的计算（结构优化后），记录总能量 E_sol
2. 运行一次 `imp_sol = 0` 的计算（相同结构优化后），记录总能量 E_vac
3. 溶剂化能 = E_sol − E_vac

> 注意：严格来说，两次计算都应在**各自优化后的结构**下取能量，因为加入溶剂后结构可能发生细微弛豫。若仅做定性估计，也可用同一结构直接比较能量。

## 4.4 结果物理解读

**为什么 E_sol_el < 0？**

溶质的电荷（电子云 + 原子核）在连续介质中诱导极化电荷，诱导极化场又反过来作用于溶质，形成吸引性的静电相互作用，降低体系总能量。这是"溶剂化使分子稳定"的主要物理机制。

**为什么 E_sol_cav > 0？**

溶质"推开"溶剂分子，形成一个空腔，需要克服溶剂内部的分子间作用力（范德华力、氢键等）。这一过程耗费能量，对溶剂化自由能有正的贡献。

**溶剂化能的净结果**：

对于极性溶质（如含极性键的分子、带电体系），`|E_sol_el|` 往往远大于 `E_sol_cav`，净溶剂化能为负（溶剂化有利）。对于非极性溶质（如 H₂），两项都较小，溶剂化能接近零。

## 常见问题 Q&A

**Q1：开启隐式溶剂后，SCF 不收敛，怎么办？**

隐式溶剂模型会改变势能面，有时使 SCF 收敛变慢。可以尝试：
- 增大 `scf_nmax`（如从 100 增到 200）
- 从真空中收敛的电荷密度出发（`init_chg = file`），再开启溶剂模型

**Q2：如何模拟非水溶剂？**

最简单的做法是只修改 `eb_k` 为目标溶剂的介电常数。若需定量精度，`tau`、`sigma_k`、`nc_k` 也需要针对该溶剂通过与实验溶剂化自由能对比来重新拟合。

**Q3：隐式溶剂模型能用于 LCAO 基组吗？**

可以。`imp_sol` 参数对 PW 和 LCAO 基组均适用，INPUT 的写法相同。
