# 调研汇总

## 一、RAG 检索结果

### 1.1 ELF 核心内容（来自案例原文）

**物理意义：**
- ELF ∈ [0,1]：1=完全局域（共价键/孤对电子），0.5=均匀电子气，0=离域
- 提出者：Becke & Edgecombe (1990)

**公式（自旋非极化）：**
- ELF = 1/(1+χ²)
- χ = (τ_KS - τ_vW) / τ_TF
- τ_KS = 正定动能密度，τ_vW = von Weizsäcker，τ_TF = Thomas-Fermi

**公式（自旋极化）：**
- ELF_σ = 1/(1+χ_σ²)（每种自旋分开）
- ELF_tot = 1/(1+χ_tot²)（总 ELF）

**LCAO 稳定性修正：**
- 远离原子核区域可能出现非物理的"局域化"
- ABACUS 在 χ 分子加 1e-5 小量修正（文献[2]建议）
- PW 基组无此问题

### 1.2 关键参数

| 参数 | 类型 | 说明 |
|------|------|------|
| `out_elf` | int int | 第1值：1=输出ELF；第2值：有效数字位数（默认3） |

**示例：** `out_elf 1 3` 表示输出ELF，精度保留3位有效数字

### 1.3 输出文件

| nspin | 输出文件 | 内容 |
|-------|---------|------|
| 1 | ELF.cube | 总ELF |
| 2 | ELF.cube | 总ELF |
| 2 | ELF_SPIN1.cube | 自旋向上ELF |
| 2 | ELF_SPIN2.cube | 自旋向下ELF |

## 二、案例解析

### 案例1：自旋非极化水分子（H₂O）

| 属性 | 值 |
|------|---|
| 体系 | 真空中的水分子 |
| 基组 | PW + LCAO（双算例对比） |
| nspin | 1 |
| 输出 | ELF.cube |
| 可视化 | 等高面图 + H-O-H 截面图 |
| 算例 | examples/elf/H20-pw，examples/elf/H20-lcao |

ELF.cube 头部（来自案例原文）：
```
STEP: 0  Cubefile created from ABACUS. Inner loop is z, followed by y and x
1 (nspin)
3 0.0 0.0 0.0
180 0.155556 0.000000 0.000000
180 0.000000 0.155556 0.000000
180 0.000000 0.000000 0.155556
 1 1.000000 11.967714 18.726812 8.322864   # H1
 1 1.000000 14.244302 20.657387 7.503683   # H2
 8 6.000000 13.550429 19.681072 9.139011   # O
```
→ 180×180×180 FFT格点，晶胞约 28×28×28 Bohr（约14.8 Å的真空盒）

**物理解读：**
- O-H键区ELF≈0.7-0.8（成键电子局域）
- O原子孤对电子ELF接近1（高度局域）
- 远离分子区域ELF=0

### 案例2：自旋极化BCC铁（bcc-Fe）

| 属性 | 值 |
|------|---|
| 体系 | 铁磁体心立方铁 |
| 基组 | PW |
| nspin | 2 |
| 输出 | ELF.cube, ELF_SPIN1.cube, ELF_SPIN2.cube |
| 可视化 | (100)晶面切面图，三种ELF对比 |
| 算例 | examples/elf/bcc-Fe-pw |

**物理解读：**
- 自旋向上/向下ELF分布不同（铁磁性）
- (100)面上可观察到d轨道的局域化特征
- 总ELF ≈ 自旋向上+向下的综合
